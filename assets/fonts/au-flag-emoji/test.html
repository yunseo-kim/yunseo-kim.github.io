File: AfricanUnionEmoji.woff2 (10.7KB, COLRv1 + CPAL)

Unicode Character Reference

  • Codepoint: U+1F30D (🌍 Earth Globe Europe-Africa)
  • HTML entity: &#x1F30D; or direct emoji 🌍
  • Falls back to 🌍 when font fails to load

Browser Compatibility

  • ✅ Chrome 90+
  • ✅ Firefox 105+
  • ✅ Edge 90+
  • ✅ Safari 15.4+

Older browsers will fall back to system emoji fonts.

CSS Usage

@font-face {
  font-family: 'AfricanUnionEmoji';
  src: url('AfricanUnionEmoji.woff2') format('woff2');
  unicode-range: U+1F30D;
}

.au-emoji {
  font-family: 'AfricanUnionEmoji', sans-serif;
}

/* HTML - Font loads: African Union flag | Font fails: 🌍 emoji */
<span class="au-emoji">&#x1F30D;</span>
